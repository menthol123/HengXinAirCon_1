# 恒信空调制冷中心官方网站

## 项目概述
恒信空调制冷中心是潍坊地区专业的空调服务公司，提供空调维修、安装、移机、回收等一站式服务。本项目是其官方网站，旨在展示服务内容，提高在搜索引擎中的可见性，吸引潜在客户。

## 主要功能
- 首页展示公司服务和优势
- 详细的服务项目介绍（维修、安装、移机、回收、清洗、配件）
- 关于我们页面展示公司实力
- 客户案例展示
- 联系我们页面提供多种联系方式

## SEO优化特性
- 优化关键词：潍坊空调维修、潍坊空调制冷、潍坊空调移机、潍坊空调回收、潍坊空调安装
- 完整的meta标签和描述
- 结构化数据标记（Schema.org）
- 响应式设计，适配移动设备
- robots.txt和sitemap.xml文件
- 页面加载速度优化

## 技术栈
- HTML5
- CSS3 (含Bootstrap 5框架)
- JavaScript
- 响应式设计
